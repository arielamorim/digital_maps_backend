Introduction

This api manages locations, calculate distances and working hours

Base URL: http://localhost

This documentation aims to provide all the information you need to work with Digital Maps API.

Authenticating requests

This API is not authenticated.

Endpoints

Show all locations.

Example request:
curl --request GET \
    --get "http://localhost/api/locations" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/locations"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Return locations by proximity

Example request:
curl --request GET \
    --get "http://localhost/api/proximity/locations?X=23&Y=50&hour=12%3A00" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/proximity/locations"
);

const params = {
    "X": "23",
    "Y": "50",
    "hour": "12:00",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):
